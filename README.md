## 프로젝트 개요

### 어떤 프로젝트 인가?  

- React hooks 사용하여 TODO Web  App 개발
- [데모링크](https://codesquad-react-todo.herokuapp.com/)

### 프로젝트를 통해 얻고자 한 것 

- webpack 을 통한 번들링 
- 리엑트의 렌더링과 라이프 사이클
- custom hook 을 활용한 재사용, useContext, useReducer, useState 를 활용한 적절한 상태관리 방법
- react-router 를 활용한 SPA 구현 및 History API 에 대한 이해
